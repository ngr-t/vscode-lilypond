# LilyPond Snippets

Snippets are available in LilyPond files (`.ly`, `.ily`, `.lyi`).

## Core Triggers

- `lyver`: version statement
- `lyrel`: relative block
- `lyscore`: score block
- `lystaff`: new staff
- `lyvoice`: named voice
- `lychords`: chord mode
- `lylyrics`: lyric mode
- `lyaddlyrics`: addlyrics block

## Score Setup Triggers

- `lyheader`: header block
- `lypaper`: paper block
- `lylayout`: layout block
- `lymidi`: midi block

## Musical Command Triggers

- `lytempo`: tempo mark
- `lytime`: time signature
- `lykey`: key signature
- `lyclef`: clef
- `lyrepeat`: volta repeat
- `lyparallel`: parallel voices
- `lyinclude`: include file

## Notes

- Snippets are intentionally minimal and editable; use tab stops to customize quickly.
- If a snippet body is too opinionated for your workflow, modify the corresponding entry in `snippets/lilypond.code-snippets`.
